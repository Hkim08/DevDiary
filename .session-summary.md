| `#224249` | `#334155` | `slate-700` / `border-slate-700` | Dark borders, dividers |
| `#0b1618` | `#0f172a` | `background-dark` / `bg-background-dark` | Darkest bg, modal backgrounds |

| `#90c1cb` | `#94a3b8` | `slate-400` / `text-slate-400` | Muted text in dark mode |
| `#1c363c` | `#1e293b` | `slate-800` / `bg-slate-800` | Card header bg |
| `#101f22` | `#1e293b` | via `slate-800` | Scrollbar track |

### Key Decisions
- Used Tailwind semantic classes (`slate-700`, `slate-800`, `bg-background-dark`) instead of hardcoded hex where possible
- Used CSS variable references (`dark:ring-background-dark`, `dark:bg-background-dark`) where the theme variable was appropriate
- Kept `uppercase tracking-tight` on Fraunces headings (matches original layout)
- Build output: 15 static pages, static mode, output to `dist/`
